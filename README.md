# java双人小游戏——滑稽大作战

本人无聊写的一个java小游戏，双人对战。

## 环境
jdk1.8.0_221


## 操作

**玩家1**

**W、A、S：跳跃、向左移动、向右移动**

**1、2: 换枪**

**J：射击**

------

**玩家2**

**方向键上、左、右：跳跃、向左移动、向右移动**

**小键盘1、2: 换枪**

**小键盘0：射击**
